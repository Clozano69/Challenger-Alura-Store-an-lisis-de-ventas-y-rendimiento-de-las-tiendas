# Análisis de Desempeño y Optimización de Ventas

## 📝 Descripción del Proyecto
Este proyecto consistió en un análisis exhaustivo de los indicadores clave de desempeño (KPIs) de cuatro tiendas pertenecientes al Sr. Juan. El objetivo principal fue identificar cuál de las sucursales presentaba el rendimiento más bajo para tomar una decisión estratégica de venta de activos.

## 📊 Factores Analizados
Para determinar la salud de cada tienda, se evaluaron las siguientes variables:
* **Ingresos Totales:** Comparativa de ventas brutas por sucursal.
* **Categorías y Productos:** Identificación de artículos con mayor y menor rotación.
* **Satisfacción del Cliente:** Análisis de calificaciones promedio por tienda.
* **Logística:** Evaluación del costo promedio de envío y su impacto en la decisión de compra.

## 🛠️ Tecnologías Utilizadas
* **Python:** Lenguaje principal.
* **Pandas:** Para la manipulación y limpieza de datos.
* **Matplotlib:** Para la creación de visualizaciones y comparativas gráficas.

## 📈 Conclusiones del Análisis
Tras procesar los datos, se determinó que la **Tienda 4** es la candidata para ser vendida. Los hallazgos mostraron:
1. Menores ingresos totales frente a la competencia.
2. Calificaciones de satisfacción del cliente significativamente más bajas.
3. Baja rotación de productos, lo que representa capital estancado en bodega.
